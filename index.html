To sync across devices:

  1. Export your data from this device
  2. Send the file to your other device (email, cloud storage, etc.)
  3. Import the data on your other device
📁
Drag and drop your data file here
or use the Import Data button above

ℹ️ How it works

  • Data is automatically saved on each device you use
  • Export creates a file with all your mileage and maintenance data
  • Import merges new data with existing data (no duplicates)
  • Your login works independently on each device

⚠️ Important

  • Each device stores data locally in the browser
  • Clearing browser data will delete your information
  • Regular exports are recommended as backups
  • Data files contain your entries but not passwords
